# Setup

<pre>
git clone git://github.com/Atinux/MultiBalls.git
cd MultiBalls
npm install -d
node app.js
</pre>

-> http://localhost:8080/ to access on remote

-> http://localhost:8080/dashboard.html to see all the balls connected
